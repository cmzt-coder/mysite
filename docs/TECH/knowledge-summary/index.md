# 知识汇总

本栏目用于沉淀“可复用、可检索、可持续迭代”的知识卡片与专题总结，定位为技术栏目下的通用知识库入口。

## 发布规则

- 目录结构：`docs/TECH/knowledge-summary/`
- 文件命名：英文小写 + 短横线分隔，例如 `cors-cross-origin-configuration.md`
- 新增文档后同步更新：
  - `mkdocs_parts/nav.yml`（侧边栏导航）
  - 本页“文档列表”
- 内容结构建议：以“概念 → 原理 → 方案 → 配置 → 排错 → 对比”组织，尽量使用可复制的最小示例。
