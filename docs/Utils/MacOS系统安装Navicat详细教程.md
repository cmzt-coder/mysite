# MacOS系统安装Navicat详细教程

## 一、前言

1. **软件用途** ：Navicat 是一款常用的数据库管理工具，支持 MySQL、PostgreSQL、SQL Server 等多种数据库，适用于开发、运维人员进行数据库连接、查询、管理操作，尤其适配 MacOS M 系列芯片设备。
2. **适用环境** ：MacOS 10.15（Catalina）及以上版本，包括 M1/M2/M3 等 Apple Silicon 芯片机型、Intel 芯片机型。
3. **注意事项** ：本教程包含第三方亲测下载渠道（附提取码），也可优先选择官方渠道；安装后常见的 “移至垃圾桶” 问题已提供完整解决方案，确保顺利启动。

## 三、MacOS 安装 Navicat 步骤（拖拽式快速安装）

1. **打开安装包**
   找到下载完成的 Navicat.dmg 文件（默认在 “下载” 文件夹），双击文件挂载虚拟磁盘，弹出安装窗口（窗口内包含 “Navicat Premium” 应用图标与 “应用程序” 文件夹快捷图标）。
2. **完成安装**
   按住 “Navicat Premium” 应用图标，直接拖拽至右侧 “应用程序” 文件夹快捷图标上，释放鼠标后等待复制完成（Finder 底部可查看复制进度），复制结束即完成安装。

## 四、安装后关键问题解决（“移至垃圾桶” 提示修复）

### 问题现象

双击 “应用程序” 中的 Navicat 图标时，弹出 “此 App 已损坏，将其移至废纸篓” 或 “移至垃圾桶” 提示（Mac 系统安全机制拦截非官方认证应用）。

### 解决方案（两步操作）

#### 第一步：调整系统安全与隐私设置

1. 打开 “系统偏好设置”，点击进入 “安全性与隐私”，切换到 “通用” 选项卡；
2. 在 “允许从以下位置下载的 App” 选项中，勾选 “App Store 和被认可的开发者”（默认可能仅勾选 “App Store”，需手动调整）；
3. 若此时界面显示 “‘Navicat Premium’已被阻止使用”，点击右侧 “仍要打开”（若无此提示，直接进行第二步）。

#### 第二步：终端命令解除文件隔离

1. 打开 “启动台”→“其他”→“终端”，打开终端窗口；
2. 输入命令：`sudo xattr -r -d com.apple.quarantine `（注意命令末尾有一个空格，不要回车）；
3. 打开 “Finder”→“应用程序”，找到 “Navicat Premium” 应用图标，按住图标拖拽至终端命令窗口中，终端会自动补全应用路径（路径格式示例：`/Applications/Navicat\ Premium.app`，系统自动处理路径中的空格）；
   * 示例命令：`sudo xattr -r -d com.apple.quarantine /Applications/Navicat\ Premium.app`
4. 按下回车键，终端提示输入密码（输入 Mac 登录密码，输入时不显示字符，输完直接回车）；
5. 命令执行完成后，关闭终端，再次双击 “Navicat Premium” 图标即可正常启动。
