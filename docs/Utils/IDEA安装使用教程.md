# IDEA安装使用教程

## 一、IDEA 安装

### 1. 官网下载

进入 IDEA 历史版本下载页面：[Other Versions - IntelliJ IDEA](https://www.jetbrains.com/idea/download/other.html)

> 注：新版本功能较完善，可根据需求选择；

### 2. 运行安装程序

1. 双击下载的安装包，进入 `IntelliJ IDEA Setup`向导，点击 `Next`；
2. **Installation Options**配置（关键选项）：
   1. 勾选 `Create Desktop Shortcut`（创建桌面快捷方式）；
   2. 勾选 `Update PATH Variable (restart needed)`下的 `Add "bin" folder to the PATH`（添加 bin 目录到环境变量，需重启生效）；
   3. 勾选 `Update Context Menu`下的 `Add "Open Folder as Project"`（添加 “以项目打开文件夹” 右键菜单，其余不选）；
   4. `Create Associations`勾选 `.java`、`.gradle`、`.groovy`（关联常用 Java 相关文件，`.kt`、`.kts`、`.pom`不选）；
3. 点击 `Next`，选择 Start Menu 文件夹（默认 `JetBrains`即可），点击 `Install`；
4. **重要提示** ：下载完成后先不要打开 IDEA！下载完成后先不要打开 IDEA！下载完成后先不要打开 IDEA！

## 二、激活（自行选择方式）

### 1. 第三方激活（测试用途）

1. 进入激活辅助网站：[JETBRA.IN CHECKER | IPFS](https://jetbra.in/)；
2. 选择 `Download jetbra zip`，下载 zip 压缩包（保持网站页面不关闭）；
3. 解压 zip 包
4. 进入 `scripts`文件夹，运行对应系统的脚本；
5. 等待脚本执行完成，弹出 `Done.`提示后点击 `确定`；
6. 返回步骤 2 的激活网站，复制对应版本的 IntelliJ IDEA 激活码；
7. 打开 IDEA，选择 `激活码(C)`，粘贴激活码完成激活。

### 2. 大学生免费激活（推荐）

前往[JetBrains 官网](https://www.jetbrains.com/)，使用教育邮箱注册账号，可享受免费使用权益，具体流程参考官网指引。
